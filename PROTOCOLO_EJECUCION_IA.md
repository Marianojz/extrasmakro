# HORAS EXTRAS V2 --- PROTOCOLO DE EJECUCIÓN CON IA INTEGRADA

## 1. Flujo de Trabajo

1.  Se recibe prompt estratégico.
2.  Se copia y pega en la IA integrada (VS Code / Cursor / Antigravity).
3.  La IA ejecuta cambios en archivos.
4.  Se confirma que no hay errores visibles.
5.  Se informa finalización de fase.
6.  Se solicita auditoría si corresponde.

No se modifican archivos manualmente. No se corrigen errores sin nuevo
prompt estratégico.

------------------------------------------------------------------------

## 2. Generación de Archivos

Cuando se necesiten nuevos documentos o configuraciones: - Se generan
mediante prompt. - No se crean manualmente. - No se editan manualmente.

Toda modificación debe ser trazable por prompt.

------------------------------------------------------------------------

## 3. Política de No Mezcla de Proyectos

Este proyecto es independiente. No se mezclan reglas, estructuras ni
aprendizajes de otros sistemas. No se reutilizan documentos externos.

------------------------------------------------------------------------

## 4. Principio Rector

Primero estabilidad. Luego hardening. Luego migración. Luego producción.

Nunca al revés.
